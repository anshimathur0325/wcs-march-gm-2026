Timeline of Women in Computing

1843

Ada Lovelace

Writes the first algorithm intended for a machine, becoming the world's first computer programmer.

1936

Grace Hopper

Begins working on early computing machines, later develops the first compiler for a programming language.

1960s

The ENIAC Programmers

Jean Jennings, Betty Snyder, and others program the ENIAC, one of the first electronic general-purpose computers.

1984

Radia Perlman

Invents the spanning-tree protocol (STP), which is fundamental to network bridging and the modern internet.

2012

Shafi Goldwasser & Silvio Micali

Shafi Goldwasser wins the Turing Award for foundational work in cryptography.

2020s

Modern Women in Tech

Women continue to shape AI, open-source projects, cybersecurity, and tech leadership worldwide.
